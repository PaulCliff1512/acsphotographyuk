# ACS Photography UK

Website source for the ACS Photography UK site.

## Configuration

Copy these example files before running the site locally or on hosting:

```text
config/database.example.php -> config/database.php
config/admin.example.php -> config/admin.php
config/storage.example.php -> config/storage.php
```

The real config files are ignored by Git so passwords are not published.

## Images

The `images/` and `uploads/` folders are kept in the repository with `.gitkeep` placeholder files, but the actual image files are ignored.

Back up the real photos separately.
